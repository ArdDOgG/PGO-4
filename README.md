# PGO-4 — Insurance Policy Portfolio Manager

A simple Java console application for a vehicle insurance office.
Manages client policies, calculates annual premiums and renewal forecasts, and generates portfolio reports.

## Classes

**Policy** — represents a single client policy. Stores client info, vehicle data, and risk level. Calculates the final premium and renewal offer based on business rules.

**InsuranceOffice** — holds all policies in an ArrayList. Generates reports, totals premiums, counts high-risk clients, and searches by policy number.

**Main** — entry point. Creates sample policies, adds them to the office, and demonstrates all features.
